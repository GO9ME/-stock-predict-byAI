# Stock Report Automation

This repository is used by the `ai-youtube` automation.

Daily reports are written with this layout:

- `YYYY/월요일/YYYY-MM-DD.md`
- `YYYY/화요일/YYYY-MM-DD.md`
- `YYYY/수요일/YYYY-MM-DD.md`
- `YYYY/목요일/YYYY-MM-DD.md`
- `YYYY/금요일/YYYY-MM-DD.md`
- `YYYY/토요일/YYYY-MM-DD.md`
- `YYYY/일요일/YYYY-MM-DD.md`

Each report contains:

- generation timestamp
- previous-day news summary for U.S. and Korea markets
- ranked U.S. top 10 watchlist with 5 mainstream names and 5 hidden-gem or surprise-rebound candidates
- ranked Korea top 10 watchlist with 5 mainstream names and 5 hidden-gem or surprise-rebound candidates
- news-based rationale for each idea
- risk checks
- source links
